# 🎮 AI+中国优秀传统文化 - 使用说明

## 🚀 快速开始

### 1. 启动项目
```bash
npm run dev
```

访问: `http://localhost:3000`

### 2. 进入体验
1. 点击"开始探索"按钮
2. 点击屏幕进入第一人称视角
3. 使用 WASD 移动，鼠标环顾

### 3. 探索场景
- 走近文物触发 AI 讲解
- 在对话框中提问
- 点击"切换场景"体验不同主题

## 🤖 AI 配置

当前使用 LM Studio，配置文件 `.env`:
```env
VITE_AI_API_KEY=not-needed
VITE_AI_BASE_URL=http://192.168.100.1:1234/v1
VITE_AI_MODEL=deepseek-r1-distill-qwen-7b
VITE_AI_PROVIDER=openai
```

## 🎨 添加 3D 模型

### 下载模型
1. 访问 Sketchfab.com
2. 搜索相关模型（如"chinese bronze"）
3. 下载 GLB 格式

### 放置模型
1. 将 GLB 文件放入 `public/models/`
2. 编辑 `src/data/sceneData.ts`
3. 添加模型路径到场景点位

示例：
```typescript
{
  id: 'my-artifact',
  name: '我的文物',
  position: new Vector3(0, 0, 0),
  radius: 3,
  description: '简短描述',
  aiContext: '详细知识...',
  modelPath: '/models/my-model.glb', // 模型路径
}
```
