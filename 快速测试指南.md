## 🐛 常见问题快速解决

### Q: AI 没有回复

**立即检查**:
```powershell
# 1. LM Studio 是否运行？
# 查看任务管理器或 LM Studio 窗口

# 2. 测试 API
Invoke-RestMethod -Uri "http://192.168.100.1:1234/v1/models" -Method Get

# 3. 查看浏览器控制台
# 按 F12，看 Console 标签
```

**解决方案**:
1. 确保 LM Studio 正在运行
2. 确保模型已加载（在 LM Studio 中选择模型）
3. 刷新网页重试

---

### Q: AI 回答不对题

**可能原因**:
- DeepSeek R1 模型特性（会先"思考"）
- 提示词需要优化

**解决方案**:
1. 尝试更具体的问题
2. 或切换到 GPT-OSS 20B 模型：
   ```env
   # 编辑 .env
   VITE_AI_MODEL=openai/gpt-oss-20b
   ```

---

### Q: 场景切换后卡住

**解决方案**:
1. 刷新页面（F5）
2. 清除浏览器缓存
3. 重新开始

---

### Q: 响应很慢

**正常情况**:
- LM Studio 本地推理需要时间
- 7B 模型：10-30秒
- 20B 模型：可能更慢

**加速方案**:
1. 等待推理完成
2. 或切换到 DeepSeek API：
   ```env
   VITE_AI_API_KEY=sk-5cd7f90f13a2454bab2c4f6edf4bb9b3
   VITE_AI_BASE_URL=https://api.deepseek.com/v1
   VITE_AI_MODEL=deepseek-chat
   VITE_AI_PROVIDER=openai
   ```

## 📊 性能参考

| 项目 | 指标 |
|------|------|
