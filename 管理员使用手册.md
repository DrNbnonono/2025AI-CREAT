## ➕ 添加3D模型场景点位

### 准备工作
1. **下载3D模型**
   - 访问 Sketchfab.com
   - 搜索相关模型（如 "chinese vase"）
   - 下载 GLB 格式

2. **放置模型文件**
   - 将 GLB 文件放入 `public/models/` 目录
   - 例如: `public/models/ming-vase.glb`

### 添加步骤
1. **打开添加表单**
   - 点击"➕ 添加新点位"按钮

2. **填写基本信息**
   - **ID**: 唯一标识，如 `ming-vase-1`
   - **名称**: 显示名称，如 `明代青花瓷瓶`
   - **位置**: X, Y, Z 坐标
     - X: 左右（正数向右）
     - Y: 高度（通常为 0）
     - Z: 前后（负数向前）
   - **模型路径**: `/models/ming-vase.glb`

3. **填写描述信息**
   - **简短描述**: 一句话介绍
     ```
     明代青花瓷瓶，景德镇官窑精品
     ```

4. **填写AI知识库**
   - 详细的文化知识，AI将基于此回答问题
   ```
   这是一件明代永乐年间的青花瓷瓶，来自景德镇官窑。
   
   特点：
   - 胎质细腻，釉色青白
   - 青花发色浓艳
   - 绘有缠枝莲纹
   
   历史价值：
   明代青花瓷是中国瓷器的巅峰之作...
   
   制作工艺：
   1. 选料：高岭土
   2. 成型：拉坯
   3. 绘画：钴料绘制
   4. 施釉：透明釉
   5. 烧制：1300度高温
   ```

5. **保存**
   - 点击"保存"按钮
   - 新点位立即出现在场景中

## 🎯 实践示例

### 示例：添加明代瓷器

**基本信息**:
```

名称: 明代青花瓷瓶
位置: (5, 0, -8)
半径: 3
模型路径: /models/blue-white-vase.glb
```

**描述**:
```
明代永乐年间青花瓷瓶，景德镇官窑精品
```

**AI知识库**:
```
这是一件明代永乐年间的青花瓷瓶（1403-1424年），代表了中国瓷器制作的巅峰水平。

器型特征：
- 口径适中，颈部修长
- 肩部丰满，腹部圆润
- 圈足规整，底部书写款识

装饰纹样：
- 主题纹饰：缠枝莲花
- 辅助纹饰：如意云头、蕉叶纹
- 青花发色：苏麻离青，浓艳深沉

工艺特点：
1. 胎质：选用景德镇高岭土，胎体坚致细密
2. 釉色：施青白釉，莹润如玉
3. 青花：使用进口苏麻离青料，呈色浓艳
4. 烧制：高温还原焰，温度达1300℃

历史价值：
明代永乐青花瓷是"青花瓷之冠"，在国际上享有盛誉。郑和下西洋时，青花瓷作为重要的外交礼品和贸易商品，远销东南亚、中东和非洲。

收藏与鉴赏：
真品存世稀少，多藏于故宫博物院、台北故宫等重要博物馆。民间收藏需要专业鉴定，防止赝品。

您还想了解青花瓷的哪些方面呢？
```

## 🎓 最佳实践

### AI知识库编写建议
1. **结构清晰**
   - 使用标题和列表
   - 分点说明

2. **内容丰富**
   - 历史背景
   - 制作工艺
   - 文化价值
   - 有趣故事

3. **语言生动**
   - 避免太学术化
   - 适当使用比喻
   - 结合现代生活

4. **字数适中**
   - 200-500字为宜
   - 过短缺乏信息
   - 过长影响阅读

### 模型选择建议
1. **文件大小** < 10MB
2. **面数** < 50K 三角面
3. **格式** GLB（包含纹理）
4. **质量** 清晰可辨认

## 📝 注意事项

1. **数据持久化**
   - 当前添加的点位仅临时存在
   - 刷新页面后消失
   - 需要手动保存到代码

2. **手动保存**
   - 复制表单内容
   - 编辑 `src/data/sceneData.ts`
   - 添加到对应场景数组

3. **模型路径**
   - 必须以 `/models/` 开头
   - 确保文件存在
   - 检查文件名拼写
